# CTF Hub
## Setup
1. Setup a PostgreSQL database

2. Add PostgreSQL settings to `.env`
    ```env
    PGUSER="username"
    PGHOST="host"
    PGDATABASE="database-name"
    PGPASSWORD="password"
    PGPORT=1234
    ```

3. Install dependencies
    ```
    npm install
    ```

4. Run server
    ```
    npm start
    ```
